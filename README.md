# Text Generation with LSTM 

## About
Generate text in the style of The Martian using PyTorch's LSTM 

## How to run
### On PC
- Install PyTorch CPU and Numpy
- Run main.py
### On Colab
- Clone Repo
- Upload all files in the checkpoints folder to Colab folder
- Run Colab cells
